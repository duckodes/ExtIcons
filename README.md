## 註冊副檔名圖標
* 使用方法: 開啟專案.sln，開啟Program.cs
## 更改預設副檔名、更改圖標檔案名稱
* 圖標檔案放入Resource資料夾內，IconName欄位需與圖標檔案名稱一致
```
        /// <summary>
        /// 要註冊的副檔名名稱
        /// </summary>
        private const string ExtensionName = "filexins";
        /// <summary>
        /// 圖標檔案名稱
        /// </summary>
        private const string IconName = "filexins.ico";
```
